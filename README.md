## ✅ 스터디 개요

- **스터디명:** 2025 estime algorithm study
- **진행 기간:** 2025년 8월 12일 ~ TBD
- **인증 방식:**
    - 매일 정해진 문제를 풀고 해당 날짜 디렉토리에 스크린샷을 제출합니다.
    - 디렉토리 이름은 `년-월-일` 형식으로 작성합니다. (예: `2025-08-12`)
    - 제출은 메인 브랜치에 직접 커밋하는 방식으로 진행합니다.
    - 제출 시각을 함께 기록해도 좋지만, 필수는 아닙니다.
